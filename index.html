전용 앱 설치 방법

1. 카톡으로 받으신 링크 주소를 복사하여 크롬 검색창에 붙여넣습니다.

2. 오른쪽 하단에 있는 "이동" 버튼을 누릅니다.

3. 우측 상단 점 3개(⋮)를 누릅니다.

4. 메뉴에서 '홈 화면에 추가'를 선택합니다.

5. 홈 화면에 추가 창에서 '설치' 버튼을 누릅니다.

6. 앱 설치 창에서 '설치' 버튼을 누르면 설치 완료!
